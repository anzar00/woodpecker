# Woodpecker Crafts & Interiors Website

A premium digital presence for **Woodpecker Crafts & Interiors** — a heritage lifestyle brand showcasing handcrafted furniture, bespoke interiors, artisanal décor, and natural treasures from Jammu & Kashmir and Ladakh.

---

## Vision

To build a modern, story-driven, luxury website that promotes:

- Kashmiri walnut wood furniture
- Crewel furnishings
- Carpets & rugs
- Paper Mache art
- Interior design services
- Kashmir saffron, honey & dry fruits
- Global export enquiries

---

## Tech Stack

### Frontend
- Angular (SSR / SSG enabled)
- SCSS
- Tailwind CSS (planned)

### Backend
- Node.js
- Express.js

### Database
- MongoDB Atlas (planned)

### Deployment
- Vercel (Frontend)
- Railway / Render (Backend)
